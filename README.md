# Face Recognition Attendance System

## 📌 Project Overview
The Face Recognition Attendance System is a Python-based application that automates attendance management using facial recognition technology. It captures and recognizes faces in real time through a webcam and records attendance automatically, reducing manual effort and improving accuracy.

---

## ✨ Features
- User Registration with Face Capture
- Real-Time Face Detection
- Face Recognition using OpenCV
- Automatic Attendance Marking
- Attendance Stored in CSV File
- Simple and User-Friendly Interface

---

## 🛠 Technologies Used
- Python
- OpenCV
- Tkinter
- NumPy
- Pandas
- Haar Cascade Classifier

---

## 📂 Project Files
- `app.py` – Main application
- `register.py` – Register new users
- `train_model.py` – Train the face recognition model
- `recognize.py` – Recognize faces and mark attendance
- `attendance.py` – Attendance management
- `attendance.csv` – Stores attendance records
- `requirements.txt` – Required Python libraries

---

## 🚀 Installation

1. Clone the repository:
```bash
git clone https://github.com/Sravs-2829/FaceRecognitionAttendanceSystem.git
```

2. Install the required libraries:
```bash
pip install -r requirements.txt
```

3. Run the project:
```bash
python app.py
```

---

## 💻 How It Works

1. Register a user's face.
2. Train the face recognition model.
3. Start the recognition system.
4. Detect and recognize faces through the webcam.
5. Automatically record attendance in the CSV file.
